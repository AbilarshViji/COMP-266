CPU

The CPU or central processing unit is like the brain of the computer. It is one of the most important parts of the computer. There are 2 main manufacturers for CPUs: Intel and AMD. Recently AMD CPUs are becoming a much better value compared to Intel CPUs. The main factors in choosing a CPU are the frequency the CPU runs at, measured in GHz, and the number of cores the CPU has. Typically the frequency decreases as the cores decreases, but recently there have been exceptions to this rule. Depending on the applications you use, some will prefer frequency, while others will prefer multiple cores.
